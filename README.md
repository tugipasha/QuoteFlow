# QuoteFlow

Basit tek‐form teklif oluşturma aracı. Frontend MVP; PDF, AI entegrasyonu (OpenRouter), influencer/freelancer preset’leri, müşteri hizmetleri sohbeti.

## Çalıştırma
- Statik dosyaları bir web sunucusuyla servis edin (ör. `npx serve` veya VS Code Live Server).
- AI için OpenRouter anahtarınızı create sayfasındaki “AI ayarları” ile girin.

## Dizinler
- `backend/` (opsiyonel MVP DB — şu an kullanılmıyor)
- Kök dizin: statik HTML/JS/CSS.

## Güvenlik
- API anahtarlarını tarayıcıda üretime almayın; proxy kullanın.
